# راهنمای آپلود پورتفولیو — GitHub + Vercel

---

## مرحله ۰ — نصب ابزارها

### ۱. Git
- از [git-scm.com/download/win](https://git-scm.com/download/win) دانلود و نصب کنید
- بعد از نصب، **Cursor** و **ترمینال** را ببندید و دوباره باز کنید

> ⚠️ روی سیستم شما قبلاً Git نصب نبود؛ بدون این مرحله بقیه مراحل کار نمی‌کند.

### ۲. Node.js
- احتمالاً دارید (چون `npm run dev` کار می‌کند)

---

## مرحله ۱ — ساخت اکانت GitHub

1. بروید به [github.com](https://github.com)
2. **Sign up** کنید (رایگان)
3. ایمیل را تأیید کنید

---

## مرحله ۲ — ساخت Repository در GitHub

1. بعد از ورود، روی **+** بالا بزنید → **New repository**
2. تنظیمات:
   - **Repository name:** مثلاً `portfolio` یا `artflow-portfolio`
   - **Public** باشد
   - تیک **Add a README** را **نزنید**
3. **Create repository** را بزنید
4. صفحه‌ای با دستورات می‌بینید — فعلاً باز بماند

---

## مرحله ۳ — آپلود پروژه به GitHub

ترمینال را در Cursor باز کنید (`Ctrl + ~`) و این دستورات را **یکی‌یکی** اجرا کنید:

</div>

```powershell
cd "C:\Users\Donya Afzar\Desktop\nemone"
```

```powershell
git init
```

```powershell
git add .
```

```powershell
git commit -m "Initial commit - ArtFlow Studio Portfolio"
```

```powershell
git branch -M main
```

```powershell
git remote add origin https://github.com/USERNAME/portfolio.git
```

<div dir="rtl" align="right">

> به‌جای `USERNAME` نام کاربری GitHub خودتان را بگذارید.  
> به‌جای `portfolio` نام repository که ساختید.

</div>

```powershell
git push -u origin main
```

<div dir="rtl" align="right">

- از شما **username** و **password** می‌خواهد
- برای password باید از **Personal Access Token** استفاده کنید (نه رمز عبور GitHub)

### ساخت Token (اگر لازم شد)

1. GitHub → **Settings** → **Developer settings** → **Personal access tokens** → **Tokens (classic)**
2. **Generate new token**
3. تیک **repo** را بزنید
4. Token را کپی کنید و به‌جای password در `git push` بزنید

---

## مرحله ۴ — Deploy روی Vercel

1. بروید به [vercel.com](https://vercel.com)
2. **Sign Up** → **Continue with GitHub**
3. به Vercel اجازه دسترسی به GitHub را بدهید
4. **Add New…** → **Project**
5. repository پورتفولیو را پیدا کنید → **Import**
6. تنظیمات (معمولاً خودکار درست است):
   - **Framework Preset:** Next.js
   - **Root Directory:** `./`
   - **Build Command:** `npm run build`
7. **Deploy** را بزنید
8. حدود ۱–۲ دقیقه صبر کنید

---

## مرحله ۵ — لینک برای مشتری

بعد از deploy موفق، لینکی شبیه این می‌گیرید:

</div>

```
https://portfolio-username.vercel.app
```

یا:

```
https://artflow-portfolio.vercel.app
```

<div dir="rtl" align="right">

همین لینک را برای مشتری بفرستید.
